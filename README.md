# Volley Rio ⚡

A volleyball match coordination app for Rio de Janeiro.

## Features

- **🗺️ Map View** — See all active volleyball matches on a map of Rio
- **📋 Matches List** — Browse available matches with player counts and times
- **➕ Create Matches** — Post new matches for other players to join
- **👥 Player Coordination** — Join matches and see who's playing

## Getting Started

This app is built with React Native and runs on Expo Snack.

### Quick Start

1. Go to [Snack Expo](https://snack.expo.dev)
2. Import this GitHub repo: `LetsVolleyBro/volleyball-app`
3. Scan the QR code with your phone (iOS: Camera app, Android: Expo Go app)
4. Start playing!

To run locally:

```bash

```

Then scan the QR code with Expo Go on your phone.
